A React Native (Expo) mobile app that allows residents in large complexes/flats to share items by creating and browsing item listings.
Users can sell, rent, donate, or request items and upload photos stored securely in Firebase Storage.
Each listing includes details like title, category, price, description, and delivery options.
The app integrates Firebase Authentication and Firestore for user and listing management.
Built with TypeScript, Node.js, React and modern UI components for a smooth mobile experience.
